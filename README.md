# EWG Emailify Email Template

This repository contains the MJML and HTML code, exported from Figma, for EWG's email template built with Emailify.

## Files
- `index.mjml`: The MJML code for the email template.
- `img/`: A folder containing any images used in the email template.

## Usage
1. Take all images in the `img/` folder and upload them to the `emailify` folder in EveryAction.
2. Copy the HTML code from `index.html` and paste it into the HTML editor in EveryAction when creating your email campaign.

Alternatively, you can use the MJML code in `index.mjml` to make any necessary adjustments before exporting it to HTML. Use https://mjml.io/try-it-live to edit the MJML and convert it to HTML then copy the resulting HTML code into EveryAction.
